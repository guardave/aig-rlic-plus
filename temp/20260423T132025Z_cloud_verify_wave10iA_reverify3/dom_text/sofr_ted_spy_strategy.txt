How the Signal is Generated
Compute the SOFR-TED spread (SOFR − DTB3) daily.
Compute its 63-day rate of change: roc_63 = spread_t / spread_{t-63} − 1.
Fix the 25th-percentile threshold on the in-sample window (2018-04 to 2022-12).
Each day, compare today's roc_63 to that fixed threshold.
If below the threshold (stress easing), enter Long SPY with a 10-day lead lag; otherwise hold cash.

How to Use This Indicator Manually

A human operator wanting to replicate the rule can:

Pull the latest SOFR and DTB3 values from FRED.
Update a rolling 63-day RoC series.
Compare today's value to the fixed 25th-percentile benchmark (published in the methodology appendix).
If below, plan a 10-business-day forward switch into SPY; if above, plan the switch into cash.
Review monthly — annual turnover is low (~23 trades), so the rule does not require daily monitoring.
